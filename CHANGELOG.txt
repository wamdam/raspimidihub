RaspiMIDIHub Changelog

User-visible changes. Short tagged one-liners, no code or commit
hashes. Tags: Added, Improved, Fix, Changed, Removed.


Unreleased

- Added: Controller - Mixer 8: drop pad row above the grid; long-press (500 ms with progress ring) captures the current state of every cell; short-press snaps every cell back to the captured state and re-emits the corresponding CCs in one go
- Added: Controller - Mixer 8 plugin: 8 knobs / 8 vertical faders / 8 mute buttons, OUT port emits CC 16-39 ch 1, IN port silently mirrors external CCs back into the on-screen cells (no feedback loops)
- Added: XY Pad control - square touch surface with a draggable dot; emits two values (x, y) for use as a dual-CC source
- Added: LayoutGrid - fixed-position param container that lets a plugin declare cells in explicit (col, row) slots with multi-cell spans; ground floor for the upcoming Controller plugin
- Improved: Plugin controls - multi-touch fixed; two fingers on different knobs or wheels now track independently instead of both following the first finger
- Improved: Panic - momentary trigger button that flashes red when fired and clears itself when the all-notes-off completes (replaces the latching toggle)
- Improved: Switch-style toggles unified into LED buttons - same behaviour, color-coded (green = on, red = destructive), more compact in dense grids
- Added: Plugin panel - section titles can be hidden for tighter dense layouts
- Added: Knob control - round body with rotating indicator, LED arc that lights up to the current value, value text in the centre; drag sensitivity scales with the value range so labelled / few-step knobs aren't twitchy
- Added: Clock Divider plugin - emits one MIDI Clock for every N received (2..32); passes notes / CC / etc. through unchanged
- Added: UI Demo plugin - drop-in showcase of every UI control type, makes no MIDI
- Added: Panic - second tap upgrades to "All Sound Off" (Elektron-style); first tap leaves delay / reverb tails ringing
- Added: Matrix clock indicator - pulses in time with the source's quarter notes instead of running a generic loop
- Added: Param span - any wheel / knob / fader / toggle / button can declare it occupies 1, 2, 3 or 4 cells of the row (e.g. a fat "Master" fader spanning the row)
- Improved: Plugin panel - 4-column responsive grid that fits even on a 360 px phone; controls share a uniform 1u footprint so a row of mixed wheels, faders, knobs, toggles, buttons all line up
- Improved: Plugin panel - cards have a brushed-metal gradient that reads as raised housings against the matrix-coloured panel background
- Improved: Meter display - now a read-only green-LED knob instead of a flat bar
- Improved: Toggles + buttons - slimmer 44 px height to match the wheel / fader visual weight
- Improved: Plugin controls - drag and scroll feel smoother; the wheel / fader / knob thumb no longer jitters back to a stale value mid-move
- Improved: Plugin controls - param values now stay in sync across all open browsers / phones via a self-healing watchdog (handles fast drags, rate limits, and dropped events)
- Improved: MIDI Delay - Delay (ms) is now a Fader; sync rate covers 4 bars down to 1/16 with triplets
- Improved: Disconnect - destinations get a clean Note Off when a connection is removed mid-note, so notes don't stick
- Improved: Load Config - reapplies only the connections that actually changed; unrelated routing keeps flowing without interruption
- Improved: CC LFO - slow clock-sync rates added (up to 8 bars)
- Improved: Multi-tab support - several phones / browsers can be open at once without the SSE connection cap pushing one of them out
- Fix: Clock Divider - silent when nothing is wired to it (was ticking from any clock the system happened to see)
- Fix: Clock Divider - dividing by 4 or higher now actually divides correctly
- Fix: Plugin panel - turning a wheel / knob no longer slides the panel away
- Fix: Radio buttons - the active pill no longer shifts its neighbours by 1 px when toggled
- Fix: Note wheel - active value now centred on the indicator
- Fix: WiFi AP - service restart now heals "no AP" / "two hostapds racing" states (was sometimes leaving the SSID dead or letting clients connect-then-immediately-disconnect)
- Fix: WiFi AP - first hostapd spawn that loses the race against NetworkManager releasing wlan0 (boot, or switching client → AP) now retries automatically; hostapd's stderr is logged on failure instead of dying silently

2026-04-11 - Version 2.0.0

Major release introducing the virtual instrument / plugin system.

- Added: Plugin system - 12 built-in plugins routable as MIDI devices in the matrix
- Added: Arpeggiator plugin - pattern player with step sequencer, accents, gate control, transport sync
- Added: CC LFO plugin - sine / triangle / square / saw / sample-and-hold with live oscilloscope
- Added: CC Smoother plugin - jitter removal with configurable smoothing factor and dual scopes
- Added: Chord Generator plugin - input note triggers full chords (major / minor / 7th / custom intervals)
- Added: Master Clock plugin - internal BPM clock with start / stop / pause and a beat meter
- Added: MIDI Delay plugin - note delay with feedback repeats and velocity decay
- Added: Note Splitter plugin - keyboard split at a configurable note into two channels with per-zone transpose
- Added: Note Transpose plugin - semitone shift up or down
- Added: Panic Button plugin - All Notes Off + All Sound Off on every channel
- Added: Scale Remapper plugin - quantize notes to musical scales with a labeled root-note wheel
- Added: Velocity Curve plugin - drawable 128-point response curve
- Added: Velocity Equalizer plugin - normalize velocity to a fixed value or compressed range
- Added: Plugin config panel - declarative UI with Wheel, Fader, Radio, Toggle, StepEditor, CurveEditor, Button, NoteSelect, ChannelSelect, Group, Display
- Added: Plugin CC automation - hardware CCs control plugin params, UI animates in real time
- Added: Plugin clock sync - 24 PPQ tick distribution with musical divisions and Start / Stop / Continue transport
- Added: Plugin display outputs - oscilloscope and meter inline in plugin panels
- Added: Plugin help - "?" button surfaces per-plugin help text
- Added: Routing matrix - plugin icons, rate meters, "Add Plugin" button
- Added: Mapping editor - Wheel / Fader / Radio / Toggle controls replacing dropdowns, with MIDI Learn
- Added: Presets - now include plugin instances and parameter values
- Added: Settings - system info, CPU and memory load, PWA install button, reload button
- Added: Haptic feedback - tick / thud sounds and vibration on the controls
- Improved: SSE - idle-connection timeout prevents stuck connections
- Improved: Test sender - direct addressing (no broadcast)
- Fix: Clock bus - auto-start, monitor-port-only counting, queue flush on Start
- Fix: MIDI Delay - circular buffer no longer explodes thread count
- Fix: Hanging notes - stable refs in piano touch handlers
- Fix: Filter engine - per-connection write ports prevent event leaking
- Fix: ALSA - correct event-type constants for Start (30) and Stop (32)
