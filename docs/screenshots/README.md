# Screenshots

These screenshots are used in the main README and docs.

## Regenerating

Captured using Playwright MCP via Claude Code at 390x844 (iPhone 14 viewport).

### Steps

1. Pi running at `http://10.1.1.2` with MIDI devices connected
2. Create temp instances of all plugin types via API
3. Capture each page/panel, then delete temp instances
