# Screenshots

Captured at 480x960 (phone-ish viewport, retina @2x) by
`scripts/screenshots/run.py`. Regenerate with `make screenshots`
against a running Pi -- the harness recreates a curated demo set
of plugins and walks every scene below. The Pi's saved config is
not touched; press **Load Config** in Settings to restore it.
